# Tic Tac Toe Game 🎮

A modern **Tic Tac Toe** game built with **HTML, Tailwind CSS, and JavaScript**.

---

## 🚀 Features

- Clean, responsive UI with **Tailwind CSS**
- Two-player game (X and O)
- Winning highlight and draw state
- Restart button for quick play
- Smooth hover & click effects
- Professional color theme

---

## 💻 How to Play

1. Open `index.html` in a browser
2. Click on any empty cell to place your mark (X or O)
3. First player to align 3 marks wins
4. Draw is highlighted in yellow
5. Press **Restart** to reset the board
